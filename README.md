# RCP Digital — Project scaffold (React + Vite + Bootstrap 5)

Estructura mínima para tu portfolio de RCP Digital. Está preparado para que lo desarrolles localmente.

## Instrucciones rápidas

1. Descomprime el zip y entra en la carpeta del proyecto.
2. Ejecuta `npm install` para instalar dependencias.
3. Ejecuta `npm run dev` para arrancar el servidor de desarrollo (Vite).
4. Abre `http://localhost:5173` en tu navegador.

**Nota:** Bootstrap se incluye por CDN en `index.html`. Cambia `logo.png` por tu logo real en la raíz del proyecto (reemplaza /logo.png).

## Estructura

- src/components: Navbar, Hero, Servicios, Portfolio, Contacto, Footer.
- src/assets: logo y demos (coloca aquí tus imágenes).
